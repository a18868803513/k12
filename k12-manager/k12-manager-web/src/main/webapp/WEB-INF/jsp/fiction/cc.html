第一章 泰山之巅

无疆 小刀锋利 3457字 2017.05.10 10:14

　　岱宗夫如何，齐鲁青未了。

　　造化钟神秀，阴阳割昏晓。

　　荡胸生层云，决眦入归鸟。

　　会当凌绝顶，一览众山小。

　　泰山。

　　龙脉汇聚之地。

　　物华天宝，人杰地灵。为五岳之首，是自古帝王封禅之地。

　　泰山一脉，绵延千里，气势恢宏磅礴。

　　虽是夜里，纵然没有月色，但却依然有种雄浑气息扑面而来。

　　那是浓郁至极的灵气！

　　玉皇顶，泰山之巅。

　　一道身影静静的站在玉皇庙前的观日亭中，面朝东方而眺，此时夜色如水，这道身影也几乎完全融入进这夜色当中。

　　天地间一片静谧。

　　民间自古以来，就有种说法，泰山不登顶，尤其对年轻人来说，寓意不好。

　　不过这种说法在如今已经有些过时了。

　　这是公元二零五六年，新纪元第三十三年。

　　其实过不过时的，对楚羽来说也是无所谓的。在很多人眼里，他的人生已经可以看到尽头了。

　　楚羽很年轻，并没有得绝症。他今年才二十出头，相貌十分俊朗，剑眉星目，身材挺拔颀长。走到哪都是一个回头率极高的大帅哥。

　　而且和那些小鲜肉不同的是，他的身上，带着一股蓬勃的阳刚之气。

　　只是此刻，他的脸上，却带着几分淡淡的忧郁气息。

　　他双目微闭，便可清晰的感受到身体内部的状态——两百四十个已经被他打通的穴位里面，蕴藏着难以想象的恐怖力量。

　　但却被一道道淡淡的紫气给封印着！

　　宛若被冰封的大海，纵然可以挥出惊涛骇浪，但在此时，也只能平静的蛰伏。

　　那紫气若隐若现，宛若明灭的云霞，虚无缥缈，又如潜龙在渊，盘旋在他体内，化成一道枷锁，遍布于他的经脉，将他身体当中所有穴位……死死封住。

　　这是一种不可思议的封印！

　　无比的神奇！

　　但对楚羽来说，却是该死的封印！混账的封印！王八蛋的封印！

　　他原本应该有另一种人生的，可以年少成名，受亿万人敬仰；可以光芒万丈，盖压同代天骄！

　　但却因为这封印的存在，迫使他不得不低着头，弯下腰，走上另一条路——如同新纪元之前，和万千学子一样，老老实实去上学，当个祖国的小花朵。

　　从经脉被封印的第一天起，到今天，已是十六年过去。

　　如白驹过隙。

　　幸亏楚羽有着豁达的心胸和乐观的心态，不然，他可能早就抑郁了。

　　关于这道封印，从小到大，楚羽用尽办法，都无法得到半点有用的信息和线索。

　　明天之后，我就要去龙城了，或许此生，再听到她的消息，就只能是在各大媒体上了吧？楚羽脸上，露出一抹惆怅。

　　如果我的体内，没有这该死的封印，如果我是一个年轻的绝世天骄，那她还会依然只是把我当成最好的朋友，或者兄弟么？

　　他轻声自语：“早知如此，当年真的不该好奇的……”

　　楚羽说着，伸出手，揉了揉眉心。那里，看上去跟其他人并没什么不同，但却隐藏着一个惊天的秘密。

　　“你倒是有几分自知之明，可惜，晚了。”夜色中，传来一道戏谑的声音。那声音虽然不大，但却清清楚楚的传进了楚羽的耳中。

　　“嗯？”楚羽微微皱眉，朝着发出声音的地方看去。尽管此时漆黑一片，伸手不见五指，是黎明前最黑暗的时间。但对楚羽来说，黑暗从来不是他的障碍。

　　他的力量虽然被封住，但他的一双眼，却是比正常的冲穴境八段高手还要锐利。

　　更别说他眉心处，还隐藏着一只别人看不见的竖眼。若是那只眼睁开，能看见的东西，会更多。

　　楚羽就曾经用竖眼看过一次林诗梦，结果当场就差点流鼻血，好几天都没睡好觉。

　　脑海中总是闪现着那白花花的一片……

　　以至于那些天他甚至完全不敢接触林诗梦，他那敏感而又脆弱的年轻身体，也跟着备受煎熬。

　　从那之后，他再不敢轻易动用眉心那只眼。

　　不过这时候，他必须要动用这只眼。

　　因为从对面走过来两道身影，身上全都散发着强烈的力量波动，宛若两只危险的野兽，正在一步步向他逼近。

　　楚羽微闭双目，眉心那只看不见的竖眼瞬间张开，射出两道肉眼无法捕捉的光芒，直指来人本源！

　　对方身体中的经脉分布，在楚羽这只竖眼之下，洞若观火！楚羽可以很清晰的看见这两人身体中那些已经开启的穴道！

　　冲穴境五段！

　　冲穴境六段！

　　楚羽有些震惊，这是两个真正的高手！

　　在当下这个时代，能够修炼到冲穴境三段以上的人，就已经算是强者了。拥有强大的武力，一拳击碎大石，一掌斩断大树，都是小意思。

　　达到五段六段的武者，更是强大得不像话！

　　“杀你的人。”走在前面的，就是刚刚出言戏谑楚羽的人，冲穴境五段。他看上去三十几岁，身材很匀称，相貌普通，属于那种扔到人堆里很难辨认的类型。

　　另一个人，则有四十多岁，三角眼，满脸横肉，看上去就凶巴巴的，他是那个冲穴境六段的高手。

　　“你们是……杀手？”楚羽皱眉，心里多少有点后悔，大意了。平日里，他的身边一直都有两个保镖在暗中保护他的。但这次，他想办法甩掉了那两个保镖，偷偷跑出来。想要在接受家族的安排之前，任性一次。没想到，就这一次，就出了意外。

　　这是新纪元第三十三年，也是北地楚家入世的第三十三个年头。这些年来，楚家的对头可不算少。

　　不过一来楚羽平素非常低调，哪怕是燕京大学的同寝室同学，也没有几个知道他真正身份的；二来知道他身份的人，都很清楚他是一个不能修炼的废物。

　　冲穴境八段的境界，也唯有他自己才知道。就连他的父母亲人，都不知道！

　　对付一个废物，需要派出两个这种级别的高手么？再者，杀了他之后，北地楚家绝对会无比的震怒，会展开疯狂的报复。

　　楚家在华夏被称为北方狼族！那可不是说着玩的。

　　楚家子弟，充满血性，一旦招惹，不死不休！

　　这是楚家的家风！

　　楚家内部虽然偶有小纷争，但一旦涉及到对外，绝对会瞬间抱成一团。

　　现如今整个华夏，有胆子直接击杀楚家嫡子的势力，还真不多。

　　“杀手？”相貌普通的人笑笑：“就算是吧。”

　　楚羽看着他们两人：“你们会不会搞错了？认错人了吧？我就是一个普通人。”他试图讲道理。

　　“认错人？不不不，我们很专业的！”

　　相貌普通的人连连摇头，如数家珍的说起来：“楚羽，生于新纪元元年入世的北地楚家，是七爷楚天北的二公子。你生下来就是一个天才，三岁自行开启三个穴位，可以举起百斤石锁，被誉为拥有罕见天赋的神童；六岁开启三十三个穴位，踏入冲穴境一段。号称神童、绝世天骄。传说有踏入冲穴境大圆满境界的希望。在你六岁就成为冲穴境一段武者那年，整个北地楚家的族人一共受到过一千二百二十六次刺杀，其中有一千一百二十八次……是针对你来的。”

　　楚羽摸了摸鼻子：“真有那么多次？还这么精准？不是你信口胡诌的吧？”

　　“哼！”相貌普通的人脸上露出一丝骄傲之色，不屑的说道：“我需要胡诌？这可是……”

　　说到这，相貌普通的人嘿嘿冷笑起来，转移了话题：“不过，也正是那一年，不知什么原因，你突然间废掉了！一开始所有人都以为这是楚家想要韬光养晦，把你隐藏起来。但一年之后，终于确认，你是真的废掉了。真是可惜啊！如果你没有废掉的话，恐怕今天至少也会像我一样。呵……二十多岁的冲穴境五段高手，甚至比我还厉害，达到六段、七段！那样的话，想要杀你，还真得费一番周折。”

　　这时候，那个凶悍的三角眼突然开口：“左宗，夜长梦多，别废话了，赶紧动手，割了他的脑袋走人。”

　　左宗？要是再加上一个棠字，倒是一个晚清的名人了。楚羽将这个名字记在脑子里。

　　“好歹也是大家族的公子呢，我还第一次杀身份地位如此尊崇的人，心情略激动！嘿嘿，盗亦有道，叫他做个明白鬼，以后黄泉路上不要怪罪我们兄弟。”左宗笑眯眯的说道。

　　“言多必失。”三角眼提醒。

　　“好吧。”左宗耸耸肩，抬头望向楚羽：“抱歉了，楚羽同学……你该上路了！”

　　说话间，左宗从腰间抽出一把尺许长的短刀。

　　“正宗尼泊尔军刀，嗷嗷快，保证让你没有任何痛苦。”左宗像是在念广告词，但他的眼神中，却透出一股子冰冷嗜血的光芒，还带着一点点兴奋。

　　这绝不是一个什么话唠，这是一个杀人如麻的冷血杀手！

　　“等等……等等！”楚羽看着两人大叫起来。

　　“怎么？想求饶？想知道幕后主使者？还是想要出双倍的价钱买命？省省吧楚公子，没意义的。为了杀你，我们可是大费周章啊！”左宗拎着那把尼泊尔军刀，一脸嗜血的看着楚羽。

　　“我可以报警么？”楚羽拿出手机，冲着两人晃了晃：“可以打幺幺零不？”

　　“报警？幺幺零？哈哈哈……孩子，你电视剧看多了吧？”左宗的眼中，露出一抹强烈的戏谑之色：“你知道为什么今晚泰山上没有游人么？你知道有种东西，叫做信号屏蔽器么？”

　　“我随便说说。”楚羽一呲牙，把手机又收了回去。

　　其实他刚刚是真的在打电话，用的语音拨号。幺幺零，在楚羽手机上代表的可不是警察。

　　可惜，没打通，这里的信号，的确被屏蔽掉了。

　　看得出来，对方为了杀他，确实做了很多准备。

　　“我要是死了，我的家族肯定会去调查，绝不会放过任何一个人，你们既然这么了解楚家，应该知道楚家的家风吧？”

　　楚羽一脸镇定的看着两人，侃侃而谈。其实心里已经很紧张了，但身为大族子弟，临危不惧，保持体面，是最基本的素质。

　　就算死……也得死的有尊严！
